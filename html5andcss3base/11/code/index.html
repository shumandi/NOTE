盒模型 css处理网页时，认为每一个元素都被包裹在一个看不见的盒子里。 这个盒子分为：内容区域，内边距（padding），边框（border），外边距（margin） 默认情况下width指的是内边距里面的内容区域的宽度。 box-sizing: border-box 模式下，width表示的是内容区域，内边距，边框加和的宽。
